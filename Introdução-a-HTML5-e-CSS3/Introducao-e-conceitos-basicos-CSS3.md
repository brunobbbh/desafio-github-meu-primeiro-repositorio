## ID x Classe



O elemento CSS altera qualquer elemento sendo em HTML.

<header id="header" class="header"></header>



<header class="header"></header>

class = .header {
	pading: 10px;
}

#header {
	padding: 15px;
}



O id só pode ser usado uma vez na página.
